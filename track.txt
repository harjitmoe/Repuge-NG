First model:
- Keystroke received by CLIENT.
- SERVER queries keystroke from CLIENT.
- SERVER responds to keystroke.
- SERVER requests CLIENT to redraw.
- CLIENT queries from SERVER: pt (twice)
- CLIENT moves cursor
- CLIENT queries from SERVER: grid
- CLIENT queries flat objgrid from SERVER
- SERVER flattens objgrid and returns it to CLIENT
- CLIENT draws current map

Second model:
- Keystroke received by CLIENT.
- SERVER queries keystroke from CLIENT.
- SERVER responds to keystroke.
- SERVER flattens objgrid
- SERVER requests CLIENT to redraw, passing: pt, grid, flat objgrid
- CLIENT moves cursor
- CLIENT draws current map
